# Peak Finder

## Running linter

To view linting erros across the entire project run the following command from the project's root folder:

```
npm run lint
```

To fix them run:

```
npm run lint-fix
```

## Running prettier

To view formatting errors run the following from the project's root folder:

```
npm run prettier
```

To fix them run:

```
npm run prettier-fix
```
